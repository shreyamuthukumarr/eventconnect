-> Event Connect

Event Connect is a web-based application developed for Event Proposal Management.  
The platform allows users to submit, manage, and organize event proposals efficiently through a simple and structured interface. The system integrates cloud-based database functionality to handle dynamic data storage and retrieval.

-> Features
- Submit event proposals
- View and manage submitted proposals
- Clean and responsive user interface
- Real-time data storage using Firestore
- Organized layout for easy navigation
- Dynamic content handling

-> Technologies Used
- HTML
- CSS
- JavaScript
- Firebase Firestore (Cloud Database)

-> Project Objective
The objective of this project is to simplify the event proposal process by providing a centralized platform where event details can be submitted, stored, and accessed efficiently. This project also demonstrates the implementation of frontend development combined with cloud database integration.

-> How to Run
1. Clone the repository
2. Open the project folder
3. Configure Firebase credentials (if required)
4. Open `dashboard.html` in your browser
